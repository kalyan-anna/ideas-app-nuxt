# ideas-app-nuxt


Deployed to AWS . 
http://vue-idea-app.s3-website-ap-southeast-2.amazonaws.com/


## Build Setup

``` bash
# install dependencies
$ npm run install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
